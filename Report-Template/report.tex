\chapter*{Abstract}

This project addresses the disconnect between UI design and code implementation, and the lack of intelligent knowledge retrieval in CRM platforms, within the context of SpurSol's Knoccs platform.
\\\\
For Design-to-Code, we developed an iterative instruction refinement methodology for Claude Code connected to Figma via Model Context Protocol (MCP) server. The instruction set evolved through nine iterations into a seven-phase, multi-agent pipeline with eight specialized agents and human-in-the-loop governance. Using this pipeline, 20 Angular components were generated from Figma designs, achieving an average CLIP visual similarity of 82.6\%, validated through behavioral unit testing and Storybook visual verification. A sync pipeline was built to keep components synchronized with evolving designs at 80--90\% automation, and its applicability was demonstrated through Tag and Macro Management feature implementation.
\\\\
For the Knowledge Base, a RAG framework was utilized combining BM25 keyword search, dense vector search with pgvector, and cross-encoder re-ranking, with company-level data isolation and multi-turn conversation support. The system was evaluated using DeepEval across 52 ground truth queries measuring Answer Relevancy, Faithfulness, and Contextual Precision, and was fully integrated into the Knoccs platform.
\\\\
Both pipelines were developed in collaboration with SpurSol and validated against a real production codebase.
