In this project, we developed two core pipelines for SpurSol's Knoccs CRM platform: a Design-to-Code Pipeline and a Knowledge Base using RAG framework. We detailed our methodology for iterative instruction refinement across nine instruction file iterations, multi-agent orchestration with eight specialized agents, component generation and evaluation, Figma-to-code synchronization, and RAG system design with hybrid retrieval and DeepEval-based evaluation.
\\\\
The results demonstrate the feasibility of AI-assisted design-to-code automation in a production Angular codebase: 20 components were generated at an average CLIP visual similarity of 82.6\%, computed per component by comparing one representative Storybook screenshot against the corresponding Figma frame. The RAG-based knowledge retrieval system was evaluated across 52 ground-truth queries spanning 10 companies and was integrated into the Knoccs platform behind the .NET proxy. Tag and Macro Management features were implemented end-to-end using the generated component library, demonstrating that the pipeline supports real feature development; their final rollout into the live Knoccs deployment is in progress with the SpurSol team.
\\\\
While the progress thus far is promising, several avenues remain for exploration. We intend to restructure the Knoccs design system with machine-readable semantics to reduce token usage and improve generation accuracy, and to benchmark alternative LLMs against Claude for component generation quality and cost. For the RAG pipeline, connecting to Knoccs's live production data through an automated ingestion pipeline is the most impactful next step, alongside migrating to gRPC communication to align with the existing microservices architecture. Further improvements to the embedding model, chunking strategy, and LLM selection are planned to enhance retrieval precision and response quality. Continuous evaluation against expanding ground truth datasets and rigorous validation within real client workflows will be central to the next stages of development.
